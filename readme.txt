[RU]
Данный модуль интеграции с Kaznachey.ua разработан для Drupal-ubercart 6.x-2.2.

Перед установкой модуля необходимо зарегистрироваться на сайте kaznachey.ua (раздел "Подключиться"). 
После подписания договора быдут выданы: 
-идентификатор мерчанта; 
-секретный ключ мерчанта; 

Порядок установки модуля:
1. Скопируйте папку /sites/all/modules/ubercart/payment/ вашего сайта;
2. Зайдите в панель управления вашего сайта;
3. Откройте страницу настройки модулей (/admin/modules);
4. Включите модуль Kaznachey в группе Ubercart - payment;
5. Перейдите на страницу настройки способов оплаты (/admin/store/settings/payment/edit/methods);
6. Выберете "Kaznachey.ua" и нажмите "Настройки метода Kaznachey";

7. Настройте способ оплаты "Kaznachey settings", заполнив все необходимые поля переданные при заключении договора:
	- Идентификатор мерчанта - используется для подписи сообщений при переадресации (необходимо получить у менеджера после подписания договора);	
	- Секретный ключ мерчанта - используется для подписи скрытых сообщений на Result URL (необходимо получить у менеджера после подписания договора);	

8. Сохраните изменения.
